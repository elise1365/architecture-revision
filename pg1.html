Architecture and organisation

  • Architecture = attributes that have a direct imapct on the logical execution of a program
  • Organisation = the operational unite=s and their interconnections that realise the architectural operation

Computer function

Function is the operation of each individual component as part of the structure

Computer functions:

  • Data processing
  • Data storage
  • Data movement
  • Data control

Computer structure

Structure is the way the components are interralted

There are 4 main structural componenets:

  1. CPU - controls the operation of the computer and performs data processing functions
  2. Main memory - stores data
  3. I/O - moves data between the computer and its external environment
  4. System interconnection - buses provide communication between the first 3 componenets

CPU components:

  • CU - controls the operation of the CPU
  • ALU - performs the comuters data processing functions
  • Registers - provides storage internal to the CPU
  • CPO interconnections - provides communications between CU, ALU and registers

IAS computer/ stored program concept

Memory of the IAS

1000 storage locations, each called 'words' of 40 bits

Both data and instructions are stored there

A word may also contain 2 20 bit instructions with each instruction made up of an 8 bit opcode and a 12 bit address.

Stored program concept

Machine code instructions stored in main memory are fetched and executed serially by a processor that performs arithmetic and logical operations
